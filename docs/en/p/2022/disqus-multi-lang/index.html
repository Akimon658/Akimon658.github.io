Use Disqus on multilingual Hugo site

2022-01-26Hugo

According to the Disqus help page, we can override the language by adding this.

var disqus_config = function () {
	this.language = "ja";
};

So, if you want to change the language of Disqus by the page’s, adding the following code to the disqus_config of your theme or official Disqus template should effect.

var disqus_config = function () {
	this.language = '{{ .Lang }}';
};

However, the variable name disqus_config would be changed if you used hugo --minify so you couldn’t change the language.

Against minify

One day, I realized that disqus_config should be called from somewhere because it was just declared in the above code. Therefore, I visited akimon658-githug-io.disqus.com/embed.js and found this.

var _config = window.disqus_config;

So I tried the following code, and it works correctly.

var disqus_config = function () {
	window.disqus_config = '{{ .Lang }}';
};

window objects and global variables

We can use window.foo same as a global variable foo. I didn’t know that.
